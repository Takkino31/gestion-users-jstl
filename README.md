# Gestion des Utilisateurs - ArrayList & JSP (JSTL)

Ce projet implémente une gestion simple des utilisateurs en utilisant une **ArrayList** pour stocker les utilisateurs en mémoire. Les pages JSP utilisent **JSTL** (JSP Standard Tag Library) pour afficher les utilisateurs et gérer les actions CRUD (ajouter, modifier, supprimer).

## Fonctionnalités
- Ajouter un utilisateur
- Modifier un utilisateur
- Supprimer un utilisateur
- Afficher la liste des utilisateurs

## Technologies utilisées
- Java
- JSP (avec JSTL)
- ArrayList pour le stockage des utilisateurs
- Serveur Tomcat pour déployer l'application
